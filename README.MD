# Testen von Java-REST-Clients mit Wiremock

**[Dies ist das Code-Repository für den gleichnamigen Artikel unter http://blog.jstage.de](https://blog.jstage.de/)**

## Import in Eclipse
1. Git-Projekt klonen
2. In Eclipse einen neuen Workspace öffnen
3. `File -> Import -> Maven -> Existing Maven Projects`
4. Rechtsklick auf das Projekt und `Run as -> JUnit Test` auswählen

## Import in IntelliJ
1. Git-Projekt klonen
2. `New -> Project from existing sources -> pom.xml`
3. Rechtsklick auf das Projekt und `Run 'All Tests'` ausführen
